# Instructions
When this skill is invoked:
1. **Design Tokens:** Define a strict CSS variable system for colors (60-30-10 rule), spacing (8px scale), and typography.
2. **Typography:** Avoid system fonts. Implement pair-based typography (e.g., Serif for headings, Sans-serif for body).
3. **Layout:** Prioritize CSS Grid for complex layouts and Flexbox for components. Avoid "AI-standard" centered card grids; use intentional asymmetry.
4. **Visual Polish:** Use subtle shadows (`box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1)`), rounded corners (min 8px), and consistent border-widths.
